# Convert JSON to CSV

This Python script converts a JSON file (`input.json`) into a CSV file (`output.csv`), using the JSON keys as column headers.

## Requirements
- Python 3.x

## How to Run
1. Place `input.json` in the same folder as the script.
2. Run:

```bash
python main.py
```
3. The CSV will be created as `output.csv` in the same folder.
